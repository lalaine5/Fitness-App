   📥 DOWNLOAD google-services.json - SUPER SIMPLE STEPS
═══════════════════════════════════════════════════════════════

1️⃣  OPEN FIREBASE
    → Go to: https://console.firebase.google.com/
    → Sign in with Google

2️⃣  CREATE PROJECT
    → Click: "Create a project"
    → Name: my-fitness-app
    → Click: Continue → Continue → Create project

3️⃣  ADD ANDROID APP
    → Click the Android icon (robot 🤖)
    → Package name: com.example.fitness_app
    → Click: "Register app"

4️⃣  DOWNLOAD FILE
    → Click: "Download google-services.json" button
    → File downloads to your Downloads folder

5️⃣  MOVE FILE
    → Open File Explorer
    → Go to Downloads folder
    → Find: google-services.json
    → Copy it (Ctrl+C)
    → Go to: C:\Users\narta\OneDrive\Desktop\Fitness App\fitness_app\android\app\
    → Paste it (Ctrl+V)

6️⃣  VERIFY
    → Check file is at: fitness_app\android\app\google-services.json
    → File size should be 1-2 KB

✅ DONE! File is downloaded and in the right place.
