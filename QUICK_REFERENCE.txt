🔥 FIREBASE SETUP (5 min)
  1. https://console.firebase.google.com/
  2. Create project: "my-fitness-app"
  3. Add Android app: com.example.fitness_app
  4. Download: google-services.json
  5. Enable: Email/Password auth

📁 CRITICAL FILES
  ✓ android/app/google-services.json (download from Firebase)
  ✓ lib/firebase_options.dart (update with your values)

🚀 RUN COMMANDS
  flutter pub get          # Install dependencies
  flutter run              # Run app
  flutter clean            # Fix build issues

🔧 TROUBLESHOOTING
  • Build error → flutter clean && flutter pub get
  • Firebase error → Check firebase_options.dart values
  • File not found → Verify google-services.json location
