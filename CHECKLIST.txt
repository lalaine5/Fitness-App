📋 YOUR TODO (5 minutes):

□ Step 1: Create Firebase Project
  → Go to: https://console.firebase.google.com/
  → Click "Create a project"
  → Name: my-fitness-app
  → Disable Analytics (optional)
  → Click "Create project"

□ Step 2: Add Android App
  → Click Android icon (robot)
  → Package name: com.example.fitness_app
  → App nickname: Fitness App
  → Click "Register app"

□ Step 3: Download Config File
  → Click "Download google-services.json"
  → Save to: fitness_app/android/app/google-services.json
  → IMPORTANT: Must be in android/app/ folder!

□ Step 4: Get Your Firebase Values
  → Click gear icon → "Project settings"
  → Scroll to "Your apps"
  → Copy these 4 values:
     • API Key
     • App ID
     • Project ID
     • Messaging Sender ID

□ Step 5: Update firebase_options.dart
  → Open: fitness_app/lib/firebase_options.dart
  → Find the "android" section
  → Replace:
     YOUR_ANDROID_API_KEY → (paste your API Key)
     YOUR_ANDROID_APP_ID → (paste your App ID)
     YOUR_PROJECT_ID → (paste your Project ID) [2 places]
     YOUR_SENDER_ID → (paste your Sender ID)

□ Step 6: Enable Email Authentication
  → Firebase Console → "Authentication"
  → Click "Get started"
  → Click "Email/Password"
  → Toggle ON
  → Click "Save"

□ Step 7: Run Your App!
  → Open terminal in fitness_app folder
  → Run: flutter pub get
  → Run: flutter run
  → Test signup and login!
